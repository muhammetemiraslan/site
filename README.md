## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/muhammetemiraslan/site.git
```

Gerekli paketleri yükleyin

```bash
  pip install -r requirements.txt
```

Sunucuyu çalıştırın

```bash
  python ./manage.py runserver
```
